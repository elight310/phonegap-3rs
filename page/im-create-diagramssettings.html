Diagram style settings

Some style settings such as background color and texture and link style apply to the entire diagram that you are working on. Items like fill and line color, line width, and text options can be set for selected symbols or links.

In addition, Inspiration® Maps™ has assigned certain defaults for its basic diagram document. These include symbol shape, text font, size and style, as well as items like fill and line color, and line width.  You see these defaults when you start up a new diagram document and start creating idea symbols and linking those symbols together. Sometimes you will want to change these defaults.

Change diagram background color, background texture and link style

To use:

  1. Tap  on the toolbar.
  2. Tap Diagram

  3. Tap to change the various options:
    • Tap a Link Style to change the type of link the diagram uses
    • Tap a background fill to change the color of the diagram background
    • Tap a background texture to change the texture used in the diagram background
  4. Tap  again to close and return to the diagram.

 

Set new defaults for your diagram

You change the default look of your symbols, links, and text using the  on the toolbar. 

To use:

  1. Tap to select the symbol or link to change to reflect the attributes you want to use as your default.
  2. Tap  on the toolbar.

  3. Do the following:

    • For symbols, tap Shape to change the default shape or use a graphic as the default shape.
    • Tap Style to change the symbol fill and line color or line width or link line color, line width and arrow direction.
    • Tap Text to change the selected font, its size, apply italic and bold styling and change the text color.
    • Tap Diagram to change characteristics for the entire diagram, these include link style, background fill and background texture.  Diagram changes will operate on the entire diagram regardless of what is selected.
  4. Tap Use as Default to set new defaults for symbols and links. Use as Default sets the currently selected fill and line color, line width, and text characteristics as the defaults for new symbols, links and text. If a symbol is selected, the shape of that symbol will also become the default for new symbols.
  5. Tap Apply Default to apply the default fill and line color, line width, and text characteristics to the selected symbols or links. If symbols are selected, the default shape will be applied.
  6. Tap  again to close and return to the diagram.
