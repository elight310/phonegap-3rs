Idea Symbols – add, resize, move and link together

Add a new idea symbol or subtopic symbol

Inspiration® Maps™ provides a number of ways to add idea symbols. Use one of the following:

 

  • Double tap an empty diagram space to create a new idea symbol.
  • Use  on a selected symbol.

    With a symbol selected, press and drag  out from the symbol. This will create new subtopic symbol, off of your existing one, in the location you dragged to. Or, just tap  to create a new subtopic symbol.

  • Use the RapidFire® tool to add a group of ideas quickly. This is most helpful for uninterrupted brainstorming.

    With a symbol selected, tap  on the toolbar. Type your idea into the cloud image that displays then press Next on your keyboard to add another idea. Repeat to add more ideas. Tap an open area of the diagram to complete your RapidFire entry.

Add text to a symbol

  1. Double tap inside the symbol.

This will bring up the keyboard, ready for you to start typing. For more information on editing text, see Help topic – Create Diagrams: Text- add, select, edit and style.

Add or launch a hyperlink

 

Inspiration Maps recognizes hyperlinks when you type the standard worldwide web (www.) prefix and text into a symbol. All of the text added to the symbol is then part of the hyperlink. Inspiration Maps does not support launching of hyperlinks that are in a symbol’s Note.

To add a hyperlink:

  1. Double tap inside a symbol to bring up the keyboard.
  2. Type in the hyperlink’s text starting with www. Once you finish adding text to the symbol, Inspiration Maps will change the text font to show that it recognized it as a hyperlink. 

To launch a hyperlink:

  1. Tap to select the symbol that contains the hyperlink you want to launch.
  2. Tap  to launch the hyperlink.

Select symbols

To work with a symbol or group of symbols, you often need to select them first.

Select a single symbol:

  1. Tap to select it. To deselect it, tap the symbol again.

 

Select multiple symbols:

  1. Tap each symbol to select it. Symbols stay selected until you deselect them. To deselect, tap the symbol again. To deselect all selected symbols, tap an empty diagram space.

Delete symbols

When you delete a symbol, if it has a note attached to it or links connected to it, the note and links will also be deleted.

 

To delete a symbol or symbols:

  1. Select the symbol or symbols to be deleted.
  2. Tap  to delete the items.  If you delete a symbol by mistake, tap  on the toolbar right away to restore the deleted items.

 

 

Symbol handles on a selected symbol

When a single symbol is selected you will see a group of handles surrounding the symbol.

 

Use the symbol handles to do the following:

Add a note to a symbol

  • Tap  to open the note for the symbol and then start typing. Use notes to add details about your idea.

Hide and show a symbol’s subtopic symbols.

  • Tap  to hide the symbol’s subtopic symbols. Tap again to show them.

Create a new subtopic symbol or link two symbols together, using :

  • Create a new linked subtopic symbol.

    With a symbol selected, touch  and drag your finger out from the symbol, then release the drag in an empty space on the diagram where you want to create the new subtopic symbol.

  • Link the selected symbol to another symbol to create a subtopic symbol relationship.

    To link two symbols, with the primary symbol selected, touch and drag your finger to another symbol until you touch it. When you have touched the other symbol the symbol will wiggle letting you know that you have connected with the new symbol. Release the drag to finish creating the new link.

Launch a hyperlink

  • Tap  to launch the hyperlink.  See Add or launch a hyperlink earlier in this section for information on how to add a hyperlink.

 

Move symbols

You can move single symbols or select multiple symbols to move at one time.

To move:

  1. Select the symbol or symbols to move.
  2. Drag and drop to move the symbols where you want them.

Any links connected to the moved symbols will move also, staying connected.

 

Cut, copy, and paste symbols

You can easily cut, copy, paste, and delete a symbol or multiple symbols within a diagram.

To do this:

  1. Tap to select the symbol or symbols you want to work on.
  2. Tap, hold, and then release, one of the selected symbols to bring up .
  3. Choose one of the options:
  • Cut will place the items in the iPad clipboard and remove them from the diagram.
  • Copy will place a copy of the items in the iPad clipboard and leave the original in your diagram.
  • Paste will replace your selection with what is already in the iPad clipboard.
  • Delete will delete the items from the diagram.

Manually resize a symbol

Symbols automatically resize to fit the text that you put in them. You can also manually resize any symbol using the natural pinch interface of the iPad.

To resize:

  1. Touch and hold the symbol with two fingers then drag your fingers outward to make the symbol larger or pinch your fingers together to make the image smaller. The image will grow or shrink proportionally.
